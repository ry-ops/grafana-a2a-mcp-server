# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-10-15

### Added
- Initial release of Grafana MCP Server
- Full MCP protocol implementation
- Grafana API client with async/await support
- Support for API key and basic authentication
- Dashboard operations (list, get, create, update, delete)
- Datasource operations (list, get)
- Alert operations (list, get)
- Folder operations (list, create)
- Annotation operations (create)
- System operations (health check, current user, current org)
- Comprehensive test suite
- Example usage scripts
- A2A protocol integration documentation
- UV package manager support
- Complete documentation and README

### Features
- **10 MCP Tools** exposed for Grafana operations
- **Async HTTP Client** for optimal performance
- **Type Safety** with Pydantic models
- **Error Handling** with detailed error messages
- **Flexible Auth** supporting multiple authentication methods
- **A2A Ready** designed for agent-to-agent communication

### Documentation
- Complete README with setup instructions
- A2A integration guide
- Example usage scripts
- API reference documentation
- Contributing guidelines
